# Demo Reports

This directory contains generated demo reports from running the comprehensive demonstration.

## Running the Demo

To generate a new demo report:

```bash
python3 scripts/comprehensive_demo.py
```

Or with dashboard launch:

```bash
python3 scripts/comprehensive_demo.py --launch-dashboard
```

## Demo Script

See `scripts/comprehensive_demo.py` for the main demonstration script that showcases all system capabilities.

## Reports

Demo reports are automatically generated with timestamps. Older reports are kept for reference but can be cleaned up periodically.
